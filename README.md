# Axure-template

Minimalist website landing page - Axure template.

[View deployment](https://dmitriysoym.github.io/axure-template/)

Realized adaptive for tablets and mobile devices.

## Tools used:

- SCSS preprocessor
- Flexbox
- BEM
- Interactivity is done in vanilla JavaScript.
